# What is Tutor Court?

Tutor Court is an online platform that connects students with tutors for online classes. It is a platform that helps students find tutors for online classes. The platform is built with Next.js, Video SDK, PayloadCMS, MongoDB and TypeScript.

# Understanding the project

1. both students, parent and tutors have thier own dashboard route.
2. We support both NGN & USD.
3. Tutors create a class an invite a parent and the parent can add their child to the class durring onboarding or the pparent classes page.
4. We have subjects and tutors teach those subjects by creating a class, inviting a parent or student and creating and updated the schedule for thier classes.
